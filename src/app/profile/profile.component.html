superAdmin

Special privileges:

  • Create a regularUser/groupAssis/groupAdmin/superUser
  • Delete a regularUser/groupAssis/groupAdmin/superUser
  • Create a group
  • Delete a group
  • Create a channel in a group (i.e. subgroup)
  • Delete a channel in a group
  • Add users to a group
  • Remove users from a group
  • Make a user a groupAssis of a group
  • Add users to a channel
  • Remove users from a channel
  • Unsure:
    • Can see all users of type: regularUser/groupAssis/groupAdmin/superUser
    • Can see all groups and channels

groupAdmin

Special privileges:

  • Create a regularUser/groupAssis
  • Delete a regularUser/groupAssis
  • Create a group
  • Delete a group
  • Create a channel in a group (i.e. subgroup)
  • Delete a channel in a group
  • Add users to a group
  • Remove users from a group
  • Make a user a groupAssis of a group
  • Add users to a channel
  • Remove users from a channel
  • Unsure:
    • Can see all users of type: regularUser/groupAssis/groupAdmin/superUser
    • Can see all groups and channels

groupAssis

Special privileges:

  • Create a channel in a group (i.e. subgroup) of a group (or groups) they are a groupAssis of
  • Delete a channel in a group (i.e. subgroup) of a group (or groups) they are a groupAssis of
  • Add users to a channel in a group (or groups) they are a groupAssis of
  • Remove users from a channel in a group (or groups) they are a groupAssis of
  • Unsure:
    • Can see all users of type: regularUser/groupAssis/groupAdmin/superUser
    • Can see all groups and channels

regularUser

  • Can see groups or channels user is added to
  • Cannot see groups or channels user isn't added to
